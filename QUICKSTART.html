üë• For Customers

Step 1: Login or Register

Visit the customer login page and enter credentials or create a new account

Step 2: Create a Service Request

  1. Click "New Request" from the navigation menu
  2. Select the service type (Plumbing, Electrical, Carpentry, etc.)
  3. Describe your issue in detail
  4. Provide your contact number and location
  5. Set priority level (Normal, High, Urgent)
  6. Optionally set a budget
  7. Click "Submit Request"

Step 3: Track Your Request

  • Dashboard: View request statistics and recent requests
  • My Requests: See all your requests with detailed information
  • Status Updates: Track progress from Pending to Completed

Request Status Meanings

‚è≥ Pending - Waiting for a provider to accept

‚úì Accepted - Provider has accepted your request

‚öôÔ∏è In Progress - Work is underway

‚úÖ Completed - Service has been completed

üîß For Service Providers

Step 1: Login or Register

Visit the provider login page and enter credentials or register your business

Step 2: Review Available Requests

  1. Go to your Dashboard
  2. Browse "Available Service Requests" section
  3. View customer details and request description
  4. Check location and priority level

Step 3: Accept Requests

  1. Click "Accept" button on any available request
  2. Request is now assigned to you
  3. You can now manage the request

Step 4: Manage Your Work

  1. Go to "Assigned Requests"
  2. View detailed information about each request
  3. Update status as you progress:
    • Accepted ‚Üí In Progress (when you start work)
    • In Progress ‚Üí Completed (when work is done)
  4. Add completion notes (optional)

Step 5: Build Your Profile

  • Click "My Profile" to view your business information
  • Track your completed requests
  • Monitor your rating as customers rate your service

‚ùì Troubleshooting

Can't Login?

  • Check if email and password are correct
  • Make sure you're using the right account type
  • Try refreshing the page

Data Not Saving?

  • Check if localStorage is enabled in browser
  • Ensure browser storage is not full
  • Try clearing cache and cookies

Issues on Mobile?

  • Ensure you're using a modern browser
  • Check your internet connection
  • Try rotating your device
